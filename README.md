# Revolve Agro Website

Revolve Agro is a multi-page agriculture product platform designed to showcase crop nutrition products and provide useful insights for farmers.

The platform includes product listings, product details, a dashboard for product insights, and an AI crop advisor interface.

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

---

## Features

- Agriculture product catalog
- Product detail pages
- AI Crop Advisor interface
- Product insights dashboard with charts
- Contact page for inquiries
- Responsive navigation menu
- Mobile-friendly design
